# 任务4：DATETIME时间类型

## 提交记录
`903f069` - task3-4: impl bigInt and date time types

## 问题分析
需要实现DATETIME类型（8字节），格式`'YYYY-MM-DD HH:MM:SS'`，范围1000-01-01～9999-12-31。需要验证输入合法性（月份、日期、时/分/秒范围，闰年判断等）。

关键设计：
- 存储格式：8字节int64_t位域压缩 `[second:6][minute:6][hour:5][day:5][month:4][year:14]`
- 输入通过VALUE_STRING传递，在列类型上下文中转换为DATETIME

## 实现思路

### 1. 解析与验证（`parse_datetime`）
- 格式检查：严格19字符`YYYY-MM-DD HH:MM:SS`
- 各字段范围验证：年1000-9999、月1-12、日1-31、时0-23、分0-59、秒0-59
- 闰年判断：`(y%4==0&&y%100!=0)\|\|y%400==0`时，2月有29天
- 非法输入抛RMDBError→"failure"

### 2. 位域压缩存储
```
int64_t = [second(6b)][minute(6b)][hour(5b)][day(5b)][month(4b)][year(14b)]
```
总40位，存储在8字节中。

### 3. 类型转换
- DATETIME字面量先被lexer解析为VALUE_STRING
- 在`init_raw()`/`check_clause`/`InsertExecutor`中检测目标列类型为TYPE_DATETIME时自动调用`parse_datetime()`
- 支持DATETIME↔STRING的条件比较和赋值转换
